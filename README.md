# Exoplanet Data Cleaning & Analysis

##  Project Overview
This project demonstrates a professional data cleaning workflow using *Python* and *Pandas*. I processed a dataset of over 1,000 exoplanets to prepare it for scientific analysis.

##  Data Cleaning Steps
* *Missing Data Imputation:* Handled 522 missing mass values using the *Median* to prevent outliers from skewing results.
* *Feature Engineering:* Created a discovery_era column to group discoveries by decade.
* *Text Standardization:* Cleaned the method column for consistency.
* *Validation:* Verified data types and removed incomplete records for orbital periods.

##  Technologies Used
* *Python* (Pandas, NumPy)
* *Jupyter Notebook*
* *Seaborn* (Visualization)
